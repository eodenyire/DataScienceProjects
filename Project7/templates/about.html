About Vehicle Detection & Counting Application

Welcome to the Vehicle Detection and Counting Application! This advanced web application uses computer vision and image processing techniques to automatically detect and count vehicles in images. Built with Flask and OpenCV, it provides an intuitive interface for traffic analysis and vehicle monitoring.

üîß How It Works

The application uses a multi-step computer vision pipeline:

  1. Image Upload: User uploads an image through the web interface
  2. Preprocessing: Image is converted to grayscale and filtered to reduce noise
  3. Edge Detection: Canny edge detection identifies object boundaries
  4. Contour Detection: Connected edges are grouped into contours
  5. Vehicle Filtering: Contours are filtered based on area and aspect ratio
  6. Visualization: Detected vehicles are highlighted with bounding boxes
  7. Results Display: Count and details are shown to the user
